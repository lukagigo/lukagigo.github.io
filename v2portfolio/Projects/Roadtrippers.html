Roadtrippers

This project is an experiment in user experience and interface design. The aim was to redesign the online platform Roadtrippers which is used for planning complex road trips worldwide. I have designed my own version of it’s landing page and rethought the websites main function - the process of planning a trip, since I believed there were many ways of making it more intuitive and user friendly.
